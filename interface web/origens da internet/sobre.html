Descrição do projeto

Este projeto foi desenvolvido com o objetivo de aplicar os conhecimentos adquiridos sobre HTML, compreendendo sua estrutura básica e sua importância na construção de páginas web.

Durante o desenvolvimento, foi possível entender melhor como a Internet surgiu e como o HTML se tornou uma linguagem fundamental para a criação de sites.

O trabalho também reforça a importância da organização de arquivos, uso de links entre páginas e estrutura correta de documentos HTML.

Referências (ABNT)

BERNERS-LEE, Tim. Weaving the Web. Harper, 1999.
CASTELLS, Manuel. A sociedade em rede. São Paulo: Paz e Terra, 1999.
